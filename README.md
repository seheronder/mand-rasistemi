
# 🥛 Mandıra Yönetim & Takip Sistemi

Bu proje, mandıra işletmelerinin günlük operasyonlarını (müstahsil yönetimi, süt toplama, kurye rotasyonu ve sipariş takibi) dijital ortama taşımak için geliştirilmiş **Full-Stack** bir çözümdür.

## 🚀 Özellikler
* **Kurye Yönetimi:** Kuryelerin rotalarını ve teslimat durumlarını takip edebileceği panel.
* **Müstahsil Takibi:** Süt veren üreticilerin ve borç/alacak durumlarının yönetimi.
* **Mobil Uygulama:** Sahadaki personelin siparişleri ve kategorileri anlık görebileceği kullanıcı dostu arayüz.
* **Güvenli Giriş:** Kullanıcı yetkilendirme ve login sistemi.

## 🛠️ Kullanılan Teknolojiler
* **Backend:** Node.js, Express.js
* **Frontend (Web):** HTML5, CSS3, JavaScript
* **Mobil:** React Native, Expo
* **Dosya Yönetimi:** Multer (Görsel yüklemeler için)

## 📂 Dosya Yapısı
* `/server.js`: Uygulamanın ana sunucu kodları.
* `/index.html`: Yönetim paneli ana sayfası.
* `/App.js`: Mobil uygulamanın ana giriş noktası.
